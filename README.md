📊 Трекер привычек
Простое веб-приложение для формирования и отслеживания полезных привычек за неделю и месяц
🚀 Запуск проекта
Клонируйте все все файлы проекта (index.html | app.css | storage.js | style.css | tracker.js) к себе на компьютер 
Как запустить? Просто дважды кликните по файлу index.html в проводнике и трекер запустится в любом удобном для вас браузере.
