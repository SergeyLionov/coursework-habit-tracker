📊 Трекер привычек
Простое веб-приложение для формирования и отслеживания полезных привычек за неделю и месяц. Написано на JavaScript
🚀 Быстрый запуск
Скопируйте все файлы проекта (index.html, style.css, storage.js, tracker.js, app.js, README.md) в одну папку.
Дважды кликните по файлу index.html, чтобы открыть приложение в любом браузере.
⚙️ Структура проекта
Код разделен на независимые модули в соответствии с принципом разделения обязанностей
Интерфейс:базовая разметка и стили. Поддерживает плавную смену светлой/темной темы и адаптивный дизайн.
Блок данных:отвечает исключительно за сохранение и чтение списка привычек и настроек темы в localStorage браузера.
Блок бизнес логики:ядро приложения. Вычисляет текущие серии, проценты выполнения за неделю/месяц и формирует ТОП-3 привычек с помощью методов массивов
Блок управления:связующее звено oтвечает за обработку кликов, валидацию ввода, запуск микроанимаций при выполнении привычки и генерацию модального календаря на месяц.
✨ Основные возможности
Полный CRUD-цикл: создание привычки, редактирование названия, сброс статистики, удаление.
Интерактивный мини-календарь: чекбоксы на последние 7 дней с анимацией «всплеска» при клике.
Календарь месяца: детальный просмотр всех выполненных дней текущего месяца в модальном окне.
Умная аналитика: автоматический подсчет непрерывной серии дней и процента успеваемости.
Кастомизация: переключение темной и светлой темы в один клик с сохранением выбора.
